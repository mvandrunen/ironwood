# Ironwood Crossing

2D, top-down HTML5 game prototype about rebuilding a fallen frontier town.

## Getting Started

1. Serve the folder with a local web server (e.g. VS Code Live Server).
2. Open `index.html` in your browser via `http://localhost:xxxx/index.html`.
3. Use the arrow keys to move. Walk onto the lighter green tile at (6,4) and press **E** to interact.
